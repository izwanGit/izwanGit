### 🤖 AI & Machine Learning

#### 🐻 [Beruang — AI Personal Finance App](https://github.com/izwanGit/Beruang) &nbsp; `🏆 Best System Architecture Award — FYP`

> AI-powered money management app built specifically for Malaysians. I trained a **Bi-LSTM model on 220,000+ Malaysian transaction records** (99.61% accuracy) to classify real descriptions like "Nasi Kandar" and "Grab". Powered by a RAG-based financial advisor using xAI Grok. Receipt scanning via Gemini 2.5 Flash. 13-screen React Native app with gamified Bear Evolution XP system.

**Stats:** 99.61% accuracy · 86.77 SUS Score · 98.2% of 55 testers felt more confident managing finances

`React Native` `Node.js` `TensorFlow.js` `Firebase` `RAG` `Grok API` `Gemini API` `Bi-LSTM` `Express.js`

#### 🖌️ [Batik Recognition & Classification](https://github.com/izwanGit/BatikRecognitionAndClassification) &nbsp; `🏆 Best AI Booth — AI Seminar 2025`

> Real-time computer vision system classifying Malaysian vs Indonesian batik using **Vision Transformer (ViT)** with **94.2% accuracy**. Live webcam demo with OpenCV and Flask. Roboflow-hosted inference API.

`Python` `ViT` `OpenCV` `Flask` `Roboflow`

#### 🍜 [Malaysian Hawker Food Recognition](https://github.com/izwanGit/malaysian-food-recognition)
